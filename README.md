# pi-cmux-lite

Small fork of [`pi-cmux`](https://github.com/javiermolinar/pi-cmux) that keeps only the cmux helpers I use in Pi: notifications, fresh Pi splits, and tool splits.

Included commands:
- `/cmv`, `/cmh`
- `/cmo <command...>`, `/cmoh <command...>`

Published package contents:
- `cmux-notify`
- `cmux-split`
- `cmux-open`
- `cmux-core`

This repo is currently loaded locally from `../../coding/src/pi-cmux-lite`. If Pi is already running, use `/reload`.
